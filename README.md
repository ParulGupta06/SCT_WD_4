# Task 04 – To-Do Web App

This project is a basic To-Do Web Application built using HTML, CSS, and JavaScript.  
It allows users to create and manage daily tasks easily.

## 🚀 Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Set date and time for tasks
- Store tasks using LocalStorage
- Responsive design (works on laptop & mobile)
- Simple and clean UI

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage API

## 📌 How to Run

1. Download or clone this repository
2. Open folder
3. Run `index.html` in browser
